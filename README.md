<h3>✨ Introduce ✨</h3>

- Spring Security를 이용한 사용자 인증/인가 구현 가능
- 세션 및 JWT 기반 인증 구현 가능
- OAuth2를 이용한 소셜 로그인 구현 가능
- RESTful API, GraphQL API 개발 가능
- Redis를 이용한 캐싱 가능
- Github Action, Docker, AWS EC2 이용한 배포 자동화 가능
